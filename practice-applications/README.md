# practice-applications

書籍の手順を**一から自分の手で進めるための作業用ワークスペース**です。
Hono + Cloudflare Workersの3アプリ（authorization-server / client-app / resource-server）の雛形だけが入っており、第2章のデータベース設定から自分で組み上げていきます。

「じっくり派」（設定の中身を手を動かして確認したい方）はこのディレクトリを使ってください。

```bash
pnpm install
```

以降の手順は書籍の第2章・第3章に沿って進めてください。
設定済みの状態から始めたい場合は [`initialized-applications`](../initialized-applications/README.md) を使ってください。
